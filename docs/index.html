Solar Calendar

Building the calendar

  1. List sunrise, sunset, and solar noon times. Use Python script and the 3rd party library "requests" to scrape the National Oceanic and Atmospheric Administration> for sunrise, sunset, and solar noon times for each day in 2024.
  2. List weather data. Use Meteostat, the open-source Python library that provides easy access to historical weather and climate data.
  3. Coregister the solar and weather data using timestamps. Save coregistered data as a json file. Summarize the data as 52 weeks from January through December of 2024.
  4. Prepare CAD model. Use OpenSCAD>, a programmer's solid 3D CAD modeller, to create blocks. The dimensions of which will be determined by the solar behavior and meterological readings scraped in the earlier steps.
Github code repo: https://github.com/jesnyder/solarCalendarOD
